# Sorting Photos Releases

## Main Functions:
- Copy photos and videos with sorting by creation date.
- Automatic file type detection by extension.
- Create date folders for file organization.
- Support formats: JPG, JPEG, PNG, GIF, BMP, TIF, TIFF, MP4, AVI, MOV, MKV.

## How to Use:
1. Select source folder with files.
2. Select target folder for copying.
3. Click "List Files" to preview.
4. Click "Copy Files" to start process.

## License Limitations:
- TRIAL version works for three months and allows copying maximum 10 files at a time.
- FULL version allows unlimited file copying.

# Сортировка фотографий (На русском)

## Основные функции:
- Копирование фотографий и видео с сортировкой по дате создания.
- Автоматическое определение типа файла по расширению.
- Создание папок по датам для организации файлов.
- Поддерживаемые форматы: JPG, JPEG, PNG, GIF, BMP, TIF, TIFF, MP4, AVI, MOV, MKV.

## Как использовать:
1. Выберите исходную папку с файлами.
2. Выберите целевую папку для копирования.
3. Нажмите "Список файлов", чтобы просмотреть.
4. Нажмите "Копировать файлы", чтобы начать процесс.

## Ограничения лицензии:
- TRIAL версия работает в течение трех месяцев и позволяет копировать максимум 10 файлов одновременно.
- FULL версия позволяет копировать неограниченное количество файлов.
